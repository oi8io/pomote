# Pomote v1.0.1

This release introduces Pomote's new minimalist app icon.

## What's changed

- A calm, flat icon combining Pomote's `P`, a focus cycle, and a leaf
- Complete macOS icon resources included in the application and DMG
- The icon remains crisp across Finder, Dock, and system display sizes

Pomote remains a lightweight, native Pomodoro timer for the macOS menu bar, with automatic focus and break cycles, configurable durations, and quiet reminder options.

Download `Pomote-1.0.1.dmg`, open it, and drag Pomote into Applications.

Pomote is ad-hoc signed and not yet notarized. macOS may require first-launch confirmation in System Settings > Privacy & Security.
