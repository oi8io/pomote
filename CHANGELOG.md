# Changelog

## 1.0.0 - 2026-07-20

### Features

- Native macOS menu bar Pomodoro timer built with AppKit
- Automatic focus and break cycles with configurable durations
- Focus countdown in the panel while keeping the menu bar distraction-free
- Visual, silent notification, and sound reminder modes
- Distinct focus and break colors, icons, and sounds
- Chinese and English interface with persistent settings
- Universal support for Apple Silicon and Intel Macs

### Documentation

- English and Simplified Chinese README files
- Local app and DMG build instructions
